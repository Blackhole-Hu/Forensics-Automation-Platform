# 电子取证项目

## Agent skills

### Issue tracker

使用 GitHub Issues 进行问题跟踪，不将外部 PR 作为 triage 需求来源。See `docs/agents/issue-tracker.md`.

### Triage labels

使用默认标签词汇：needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix。See `docs/agents/triage-labels.md`.

### Domain docs

多上下文布局，根目录 `CONTEXT-MAP.md` 指向各子项目的 `CONTEXT.md`。See `docs/agents/domain.md`.
