# Issue Tracker

## Platform

GitHub Issues

## CLI Tool

`gh` CLI

## Commands

- 创建问题: `gh issue create --title "..." --body "..."`
- 列出问题: `gh issue list`
- 查看问题: `gh issue view <number>`
- 关闭问题: `gh issue close <number>`

## External PRs as Request Surface

No — external pull requests are not treated as a triage request surface at this time.
